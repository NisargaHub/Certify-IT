# 📱 Certify-It – Android Based Certificate & Workshop Manager

## 📌 Overview

**Certify-It** is an Android application developed to simplify and automate the management of workshops and events. The app provides a digital platform for participant registration, attendance tracking, and automated certificate generation using Firebase as a real-time cloud backend.

---

## 🚀 Features

* 📋 Event creation and management
* 👥 Participant registration
* ✅ Digital attendance tracking
* 📄 Automated certificate generation (PDF)
* ☁️ Real-time data storage using Firebase
* 🔒 Secure and scalable cloud-based system

---

## 🛠️ Tech Stack

* **Frontend:** Android (Java/XML)
* **Backend:** Firebase Realtime Database
* **Tools:** Android Studio
* **Other:** PDF Generation Libraries

---

## 📊 How It Works

1. Organizer creates an event in the app
2. Participants are registered digitally
3. Attendance is marked within the app
4. System automatically generates certificates in PDF format
5. Data is stored and synced in real-time using Firebase

---
